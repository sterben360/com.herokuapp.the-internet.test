Тестовое задание
Описание:
Вам необходимо автоматизировать три теста в браузере (описание тестов ниже).
Желательно использовать для автоматизации браузер Chrome последней версии.
TEST-1
1. Открыть в браузере http://the-internet.herokuapp.com/
2. Перейти в Add/Remove Elements
3. Кликните на “Add Element” три раза
4. Проверьте, что появилось три элемента Delete
5. Кликните на любой Delete
6. Проверьте, что осталось только два элемента Delete
TEST-2
1. Открыть в браузере http://the-internet.herokuapp.com/
2. Перейти в JavaScript Alerts
3. Кликните на Click for JS Alert > Кликните OK на алерте
4. Проверьте, что результат You successfuly clicked an alert
5. Кликните на Click for JS Confirm > Кликните Cancel
6. Проверьте, что результат You clicked: Cancel
7. Кликните на Click for JS Prompt > Введите любое слово > Кликните OK на
алерте
8. Проверьте, что результат You entered: asd
TEST-3
1. Открыть в браузере http://the-internet.herokuapp.com/
2. Перейти в Frames > iFrame
3. Выполните нативно GET запрос
http://jsonplaceholder.typicode.com/todos?_start=0&_limit=5 и запомните все title
4. Введите все title в текстовое поле через \n (Перевод строки)
TEST-4
1. Открыть в браузере http://the-internet.herokuapp.com/
2. Перейти в File Upload
3. Выберите любой файл из ПК
4. Кликните Upload
5. Проверьте, что отображается месседж File Uploaded!
Требования:
1. Опубликовать на GitHub.
2. Использование тестового фреймворка (любого).
3. В описании на GitHub должно быть руководство к запуску вашего теста.
4. Это должен быть именно тест, а не набор скриптов. Проект должен быть
рабочим и запускаться. Для проверок используйте assert из различных
библиотек (в зависимости от выбранного языка): JUnit, TestNG, Chai, PyTest и
т.д.
5. Опционально (по желанию): Добавьте скриншоты в ваши тесты.
6. Опционально (по желанию): Добавьте возможность параллельного запуска
тестов в вашем проекте (добавьте команду или инструкцию для параллельного
запуска в описание проекта)
